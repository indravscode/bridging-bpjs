# Kontribusi

Terima kasih sudah meluangkan waktu untuk berkontribusi. Semua kontribusi sangat kami apresiasi dan terbuka untuk diterima.

## Bagaimana cara berkontribusi?

Jika kamu punya ide fitur baru yang ingin ditambahkan, atau menemukan bug yang ingin diperbaiki, ikuti langkah-langkah berikut:

- **Fork repository ini**
- **Clone repository** hasil fork dan siapkan project-nya (lihat bagian [manual installation](https://github.com/indravscode/bridging-bpjs#manual-installation) di README.md)
- **Implementasikan** perubahan yang diperlukan
- **Kirim pull request**

## Panduan

### Pesan commit Git

Ikuti spesifikasi [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/) agar pesan commit rapi dan mudah dibaca. Ringkasnya:

- Batasi subject line maksimal 80 karakter
- Huruf pertama subject line menggunakan kapital
- Gunakan present tense (mis. "Add feature" bukan "Added feature")
- Pisahkan subject dan body dengan satu baris kosong
